¿Qué es una tarjeta de sonido?

Una tarjeta de sonido o placa de sonido es un dispositivo de hadware que permite la entrada o salida de informacion en forma de sonido bajo el control de un programa informatico llamado controlador. Se usa para realizar contenidos multimedia como videos, sonidos,juegos,animaciones. A esta se pueden conectar altavoces, auriculares, micrófonos, instrumentos, etc.
